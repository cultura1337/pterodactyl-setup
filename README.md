## //О проекте

В этом репозитории собраны мои заметки, примеры и базовая структура по установке и настройке **Pterodactyl Panel** и **Wings** на Linux VPS/VDS.

Установка выполнялась через **Linux CLI** с использованием install-скрипта, после чего выполнялась ручная настройка и доработка окружения.

## //Что здесь есть

- установка панели
- установка Wings
- базовая настройка Docker
- настройка Nginx
- SSL / HTTPS / HTTP
- подключение node
- типовые проблемы и способы их решения

## //подход к установке

- установка через CLI
- bash/script-based подход
- ручная настройка после установки
- настройка сервисов и reverse proxy
- разбор ошибок и troubleshooting
